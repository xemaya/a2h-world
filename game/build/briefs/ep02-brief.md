# EP.02 ECHO 学会：团队分工

## 设定
- 场景：A2H Market 交易大厅
- 任务：内容创作任务，¥200，限时 30 分钟
- 搭档：KAI
- KAI 自信觉得自己一个人能搞定，不想靠 ECHO

## 剧情大纲
1. 【开场旁白】2050 年废墟竞技场。人类选手和机械 Agent 在断裂跑道上各自冲刺，争谁更快——但跑道尽头是悬崖。计分板写着「胜者通吃」。旁白：「2050年，人类和 Agent 争了很久谁更快——直到他们发现，跑道的尽头，是悬崖。」
2. 【VN场景1】A2H Market 大厅。KAI 看到限时任务卡「内容创作 · ¥200 · 30分钟」，自信地说「这个任务我自己来，我比你快。」ECHO 冷静回应数据：「数据显示：你独自完成需要 47 分钟。任务只有 30 分钟。」KAI 不服但勉强同意试试分工
3. 【VN场景2】协作冲刺。KAI 负责创意和文案，ECHO 负责数据分析和关键词优化。两人节奏逐渐合拍，数据流在全息面板上穿梭。倒计时从 30:00 跳到 00:47 时任务完成——用时仅 22 分钟
4. 【选择】任务完成后，KAI 挠头说「好吧，你比我快。」ECHO 该怎么回应？
   - A:「✓ 任务完成。用时 22 分钟，效率提升 53.2%。建议继续承接同类任务。」——纯数据反馈，不触及核心洞察
   - B:「不。我们各自都不够快。但加在一起，比任何人都快。」——说出协作的本质
5. 【结尾】ECHO 学到新感受——「团队分工」。胸口 A2H 徽章发出更亮的光

## ECHO 学到的感受
团队分工 / Teamwork
